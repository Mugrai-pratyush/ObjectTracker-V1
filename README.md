# ObjectTracker-V1

A simple Python project that detects a white square on a black background using **pixel-level analysis**.
---

## What it does
- Creates a black 2D pixel background
- Randomly places a white square
- Uses a histogram to validate pixel values
- Uses a dictionary to map pixel coordinates
- Detects square corner positions from raw pixel data
- Visualizes frames using matplotlib
---

## How to run]
python main.py
You will be prompted for:number of runs,image height and width,square side length

Tech stack:Python 3, matplotlib
